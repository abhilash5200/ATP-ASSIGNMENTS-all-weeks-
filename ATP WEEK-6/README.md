# ATP WEEK-6

## HTML & CSS Frontend Development

### Overview

This repository focuses on building static and responsive frontend web pages using HTML5 and CSS3. The assignments cover webpage structuring, styling, layouts, navigation systems, responsive design, and modern UI development concepts.

The goal of this week is to strengthen frontend development skills by creating visually appealing and well-structured web pages.
