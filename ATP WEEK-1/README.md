# JavaScript Practice Programs

## Overview

This repository contains a collection of JavaScript programs and assignments designed to practice core JavaScript concepts. The programs cover arrays, objects, functions, conditional statements, and commonly used array methods such as `filter()`, `map()`, `reduce()`, `find()`, and `findIndex()`.

The project is intended for beginners who want to strengthen their JavaScript fundamentals through practical examples and mini assignments.

Sample Concepts Practiced
filter()
Used for selecting elements that satisfy a condition.

let passedStudents = students.filter(student => student.marks >= 40);
map()
Used for transforming data.

let updatedMarks = marks.map(mark => mark + 5);
reduce()
Used for aggregation operations.

let total = marks.reduce((sum, mark) => sum + mark, 0);
find()
Used for locating the first matching element.

let student = students.find(student => student.marks === 92);
findIndex()
Used for finding the position of an element.

let index = students.findIndex(student => student.name === "Kiran");
